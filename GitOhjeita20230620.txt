4. kun luot uuden branchin, käyttäjien pitää ottaa se käyttöön bash-terminaalissa
git branch -m Maarittely 
(git remote set url origin https://github.com/SimpeLe/Viestittely-kansio)
git remote add origin https://github.com/SimpeLe/Viestittely-kansio
git remote -v
git fetch origin
git branch -u origin/<BRANCH> <BRANCH>
git remote set-head origin -a
git push -u origin master

TAI vaihda oletus main-branch branchiksi Maarittely näillä komennoilla
git branch -m 'Määrittely' Maarittely
git fetch origin
git branch -u origin/Maarittely Maarittely
git remote set-head origin -a
git push -u origin master
